# 📊 Stock Trading Backtest Dashboard

A Streamlit web app for backtesting stock trading strategies using Moving Average Crossover.

## Features
- Real-time stock data from Yahoo Finance
- Moving Average Crossover Strategy (SMA)
- Interactive charts with buy/sell signals
- Performance metrics (Returns, Sharpe Ratio, Max Drawdown)
- CSV data export

## How to use
1. Enter a stock symbol (e.g., AAPL, TSLA, RELIANCE.NS)
2. Select date range
3. Adjust SMA periods
4. Click "Backtest" button

## Deployed on Streamlit Cloud
